Введение в язык программирования Perl

Упражнения
I. Простейшие программы.
  1. Напишите программу, приветствующую мир.
  2. Добавьте к предыдущей программе возможность ввода объекта приветствия.
  3. Напишите программу, вычисляющую сумму, разность, произведение и частное двух введенных с клавиатуры чисел. Выведите результаты в следующей форме: "4+2=6".
  4. Добавьте к предыдущей программе проверку того, что введено именно число. Для этого можно сравнить введенную переменную с числом 0 и строкой "0".
  5. Даны две строки. Выведите длину более короткой строки (воспользуйтесь функцией length).
  6. Дана строка. Определите код ее первого символа (substr, ord).
  7. Дана строка. Выведите число 1, если строка начинается с прописной латинской буквы, после которой идет пробел.
  8. Дана последовательность чисел, заканчивающаяся нулем. Вычислите сумму всех чисел и выведите ее в следующей форме:"2+3+4+3+2=14".
  9. Выведите строку, в которой перечислены через запятую все десятичные цифры.
  10. Напишите программу, в которой загадывается число, а пользователь пытается его угадать, предлагая варианты и получая в ответ фразы типа "недолет" или "перелет".
II. Работа с массивами и хэшами.
  1. Сформируйте и выведите на консоль массив с названиями дней недели.
  2. Сформируйте хэш, содержащий количества животных в зоопарке. Посчитайте общее количество животных и среднее количество животных одного вида. Выведите упорядоченный по алфавиту список животных, присутствующих в единственном экземпляре.
  3. Дана последовательность чисел, заканчивающаяся нулем. Выведите эту последовательность в обратном порядке.
  4. Сформируйте массив чисел Фибоначчи. Выведите его на консоль (по пять элементов в каждой строке).
  5. Дан хэш, реализующий таблицу целочисленных переменных (ключ - имя переменной, значение элемента - целое число). Пользователь вводит две строки - имена переменных. Посчитайте сумму соответствующих чисел.
III. Обработка текстовых файлов.
  1. Посчитайте количество строк в заданном файле.
  2. Дан файл, содержащий по одному числу в каждой строке. Найдите среднее арифметическое всех этих чисел.
  3. Дан файл с текстом, абзацы которого разделены пустой строкой. Посчитайте количество абзацев в этом файле.
  4. Дан файл с текстом, абзацы которого разделены пустой строкой. Создайте новый файл, содержащий количества символов в каждом из абзацев исходного файла.
  5. Даны два файла. Первый файл содержит таблицу переменных с трехбуквенными именами (пример одной строки файла: "abc=2341"). Второй - несколько имен переменных (по одному имени на каждой строке). Сформируйте по первому файлу хэш, реализующий ту же таблицу переменных. Посчитайте сумму чисел, соответствующих переменным из второго файла (если значение переменной не определено, считайте ее равной нулю).

Дополнительные упражнения
I. Подпрограммы.
  1. Напишите подпрограмму, выводящую на консоль имя любого из великих математиков.
  2. Напишите подпрограмму long_print($s, $n), которая выводит строку $s на консоль $n раз. Аргументы для подпрограммы должны браться из командной строки.
  3. Напишите подпрограмму, вычисляющую гипотенузу прямоугольного треугольника по заданным катетам. Катеты должны вводиться в командной строке.
  4. Напишите подпрограмму, возводящую свой аргумент в куб.
  5. Напишите подпрограмму count_char($s,$c), подсчитывающую количество вхождений символа $c в строку $s.
  6. Напишите подпрограмму, которая формирует хэш, содержащий количество вхождений каждого символа в заданную строку.
  7. Дан массив целых чисел. Напишите подпрограмму, возводящую каждый элемент массива в квадрат и возвращающую количество ненулевых элементов массива.
II. Регулярные выражения.
  1. Посчитайте количество строк заданного файла, содержащих число 42.
  2. Удалите из данной строки все знаки препинания.
  3. На основе заданного файла сформируйте новый файл, содержащий только те строки исходного, которые начинаются с цифры.
  4. Дана строка, содержащая целые числа, разделенные знаками "плюс" (напр. "12+3737+312+12"). Найдите соответствующую сумму чисел.
  5. Определите, является ли данная строка федеральным телефонным номером компании "Мегафон" (напр. "(8-928)123-45-67").

Вспомогательные материалы
I. Подпрограммы.
Рассмотрим пример определения и использования подпрограмм в языке Perl.
#!/usr/bin/perl
hello(); #Вызов подпрограммы hello
print hello2(); #Печать результата вызова подпрограммы hello2

sub hello { #Начало определения подпрограммы hello
  print "Hello, world!\n";
} #Конец определения подпрограммы hello

sub hello2 { #Начало определения подпрограммы hello2
  return "Hello, world!\n";
} #Конец определения подпрограммы hello2
Как видно из этого примера, для вызова подпрограммы достаточно написать ее имя. Определение подпрограммы может находиться где-угодно в файле программы, независимо от места вызова. Подпрограммы могут возвращать значения, для этого можно использовать оператор return.
Подпрограмма может принимать любое количество параметров. Переданные параметры доступны в подпрограмме в виде элементов специального массива с названием "@_" (т.е. к параметрам можно обращаться так: $_[0], $_[1], $_[2],...). Часто параметры подпрограмм копируются в локальные переменные (они определяются с помощью оператора my). Обе подпрограммы в следующем примере возвращают сумму двух чисел:
#!/usr/bin/perl
print sum1(3,5);
print sum2(2,4);

sub sum1 {
  return $_[0]+$_[1]; #Вычисления производятся непосредственно с элементами массива @_
}

sub sum2 {
  my($a, $b) = @_; #Первый элемент массива @_ копируется
#в локальную переменную $a, а второй - в переменную $b
  return $a+$b;
}
Изменение переменной из массива @_ приведет к изменению значения параметра на вызывающей стороне. В следующей программе подпрограмма inc увеличивает значение переданной ей переменной на единицу.
#!/usr/bin/perl
$a=5;
print $a; #Выводится число 5
inc($a);
print $a; #Выводится число 6

sub inc {
  $_[0]++;
}
Подпрограмма может работать с массивом @_ даже не зная, сколько именно параметров ей было передано. В следующем примере определяется наибольший из переданных подпрограмме параметров.
#!/usr/bin/perl
print max(1,2,3,5,78); #Выводится число 78
print max(3, 33,2); #Выводится число 33

sub max {
  my $m = $_[0];
  foreach my $a(@_)
  {
    if($m<$a) {$m=$a;} #или $m = $m<$a ? $a : $m;
  }
  return $m;
}
Основной программе также могут передаваться параметры. Часто для этого используется командная строка. Предположим, что программа была запущена следующим образом: "./prog.pl string". Тогда строка string будет доступна в программе как первый элемент массива @ARGV (т.е. как $_ARGV[0]). При использовании командной строки для передачи параметров нельзя использовать оператор "<>" для ввода с консоли. Вместо этого нужно писать "<STDIN>" (напр. "$line = <STDIN>;").
