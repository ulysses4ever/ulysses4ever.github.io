    \[
        (\Lvar{xy} x y) yx (\Lx x)
    \]

    \[
        (\Lvar{x} y (xz)) (y ((\Lx x) y)) 
    \]
    \item Вычислить:
    $$\Lif {\isZero (2 + 3)} {((\snd (2, 3)) + 3)} {4}.$$
    Все вычисления проводятся с помощью редукции соответствующих \L-термов.
    \item Вычислить в комбин\'{а}торной логике:
    $\clS\clK\clI\clK\clS$.
