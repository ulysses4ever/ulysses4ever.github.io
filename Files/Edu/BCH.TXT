Разложим x^4 + x^3 + x^2 + x + 1 в поле F[2^4], полученном расширением F[2] примитивным полиномом x^4 + x + 1. Корни (младшим степеням полиномиального представления соответствуют левые координаты):
[1 1 1 1] = alpha^12
[0 0 0 1] = alpha^3
[0 1 0 1] = alpha^9
[0 0 1 1] = alpha^6

(x^4 + x^3 + x^2 + x + 1) - код БЧХ в узком смысле: beta = alpha^3, b = 1, d* = 5,
имеется серия нулей: beta, beta^2, beta^3, beta^4, данный многочлен - многочлен минимальной степени над F[2], имеющий такую серию нулей.
