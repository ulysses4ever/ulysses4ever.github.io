\begin{itemize}
	\item изложение BMS-алгоритма, его связи с BM-алгоритмом и возможностей его
	реализации;
	\item схему BMS-алгоритма, которая может служить практическим
	руководством к реализации;
	\item реализацию BMS-алгоритма с использованием разработанной схемы;
	\item описание АГ-кодов типа кодов Рида—Соломона и метода их 
	декодирования;
	\item схему декодера АГ-кодов типа кодов Рида—Соломона.
\end{itemize}
