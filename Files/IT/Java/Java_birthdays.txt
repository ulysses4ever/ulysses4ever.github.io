JDK 1.0 - 1996

JDK 1.1 - 1997 -
    AWT improoved, inner classes, JavaBeans, JDBC, RMI.
    
J2SE 1.2 (Playground) - 1998 - 
    'strictfp', reflection (introspection only, no modification), Swing,
    JVM was equipped with a JIT compiler, Java Plug-in,
    Java IDL for CORBA interoperability, Collections framework.
    
J2SE 1.3 (Kestrel) - 2000 -
    HotSpot JVM included,
    RMI was changed to be based on CORBA,
    JavaSound, Java Platform Debugger Architecture (JPDA),
    Java Naming and Directory Interface (JNDI) included in core libraries.    

J2SE 1.4 (Merlin) - 2002 -
    'assert', regular expressions, non-blocking NIO,
    logging API,
    image I/O API (JPEG, PNG), XML/XSLT (JAXP),
    security and cryptography extensions (JCE, JSSE, JAAS),
    Java Web Start included.
    
J2SE 5.0 (Tiger) - 2004 -
    Generics, Metadata, Autoboxing/unboxing, Enumerations, Varargs, Enhanced for loop.

Java SE 6 (Mustang) - 2006 -
    Scripting Language Support, Java Compiler API, JAXB 2.0 (including StAX parser),
    pluggable annotations,
    GUI improvements.
